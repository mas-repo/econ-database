HKDSE Economics Concept Guide

課題 Topic: 實證與規範經濟學 Positive vs. Normative Economics 難度 Difficulty: ★★☆☆☆ (2/5) 關鍵詞 Key Concepts: 實證陳述 Positive, 規範陳述 Normative, 可推翻性 Refutability

1. 題目 The Question

參閱下列彼得和瑪莉所作的陳述。
Refer to the following statements made by Peter and Mary.

彼得：香港作為一已發展經濟體，它的堅尼系數太高了。
Peter: As a developed economy, Hong Kong’s Gini coefficient is too high.

瑪莉：香港的堅尼系數是全亞洲最高的。
Mary: Hong Kong’s Gini coefficient is the highest in Asia.

_________________ 的陳述屬 ______________ 因為 ____________________。
_________________ statement is ______________ because ____________________.

A. 彼得 … 實證性 … 大部分人認同該陳述
A. Peter’s … positive … most people agree with it

B. 彼得 … 規範性 … 該陳述和現實世界的數據並不相符
B. Peter’s … normative … it is inconsistent with the real world data

C. 瑪莉 … 實證性 … 該陳述可以被事實推翻
C. Mary’s … positive … it can be rejected by fact

D. 瑪莉 … 規範性 … 該陳述涉及價值判斷
D. Mary’s … normative … value judgement is involved in the statement

2. 概念重溫 Concept Review

要拆解這類題目，我們必須清楚分辨「實證」與「規範」經濟學的根本差異。
To tackle this type of question, we must clearly distinguish the fundamental differences between "Positive" and "Normative" economics.

  • 實證性陳述 (Positive Statement): 處理「是甚麼」(What is)。這類陳述基於客觀事實，可以被事實或數據推翻 (Testable / Refutable)。注意：即使一個陳述在現實中是錯的，只要它能被驗證，它依然是實證性陳述。
    Deals with "What is". These statements are based on objective facts and can be refuted by facts or data (Testable / Refutable). Note: Even if a statement is factually wrong in reality, as long as it can be tested, it is still a positive statement.
  • 規範性陳述 (Normative Statement): 處理「應該是甚麼」(What ought to be)。這類陳述涉及價值判斷 (Value Judgements)，表達個人意見或偏好，因此不能被事實推翻 (Not Testable)。
    Deals with "What ought to be". These statements involve value judgements, expressing personal opinions or preferences, and therefore cannot be refuted by facts (Not Testable).

補充概念 (Supplementary Concept): 堅尼系數 Gini Coefficient
雖然此題不需計算堅尼系數，但我們重溫一下其定義公式：
Although this question does not require calculating the Gini coefficient, let's review its defining formula:

$$ \text{Gini Coefficient} = \frac{\text{Area A}}{\text{Area A} + \text{Area B}} $$

數值介乎 \( 0 \)（絕對平均）與 \( 1 \)（絕對不平均）之間。
The value ranges between \( 0 \) (absolute equality) and \( 1 \) (absolute inequality).

3. 答案與詳細分析 The Answer & Detailed Analysis

正確答案 Correct Option: C

Option A (錯誤 Incorrect)

彼得的陳述包含「太高」(too high)，這是一種主觀的價值判斷，因此是規範性 (Normative) 而非實證性 (Positive)。此外，「大部分人認同」並不能使一個陳述變成實證性。
Peter's statement contains "too high", which is a subjective value judgement. Therefore, it is normative, not positive. Furthermore, "most people agree" does not turn a statement into a positive one.

Option B (錯誤 Incorrect)

彼得的陳述確實是規範性 (Normative)，但選項給出的原因錯誤。規範性陳述的特徵是「涉及價值判斷」，而不是「與現實數據不符」。「與現實數據不符」描述的是一個錯誤的實證性陳述。
Peter's statement is indeed normative, but the reason given in the option is wrong. The characteristic of a normative statement is that it "involves value judgement", not that it "is inconsistent with real world data". Being inconsistent with data actually describes a false positive statement.

Option C (正確 Correct)

瑪莉的陳述指出「香港的堅尼系數是全亞洲最高」。這是一個客觀的主張，我們可以透過收集亞洲各國的堅尼系數數據來驗證這句話的真偽。因為它「可以被事實推翻」，所以它是實證性陳述。
Mary's statement claims "Hong Kong’s Gini coefficient is the highest in Asia". This is an objective claim. We can verify its truthfulness by collecting data on Gini coefficients of Asian countries. Because it "can be rejected by fact", it is a positive statement.

Option D (錯誤 Incorrect)

正如上文解釋，瑪莉的陳述可以被客觀數據驗證，並不涉及主觀的價值觀（如「好/壞」、「應該/不應該」）。「最高」(highest) 只是一個數學上的排序，不是價值判斷。
As explained above, Mary's statement can be verified by objective data and does not involve subjective values (like "good/bad", "should/should not"). "Highest" is merely a mathematical ranking, not a value judgement.

4. 題目拆解 Problem Decomposition

做這類題目時，我們需要尋找「樞紐字眼」(Pivot Words) 來快速判斷：
When doing this type of question, we need to look for "Pivot Words" to make quick judgements:

  • 彼得的關鍵字： 「太高」(too high)。甚麼數值才算「太」高？這是沒有客觀標準的，純屬個人感受。因此：彼得 = 規範性 (Normative)。
    Peter's keyword: "too high". What value is considered "too" high? There is no objective standard; it's pure personal feeling. Thus: Peter = Normative.
  • 瑪莉的關鍵字： 「最高」(highest)。這是一個統計事實的宣稱。只要把亞洲地區的數字排序列出，就能知道這句話對不對。因此：瑪莉 = 實證性 (Positive)。
    Mary's keyword: "highest". This is a declaration of a statistical fact. By listing and ranking the figures in Asian regions, we can know if this statement is true or false. Thus: Mary = Positive.

5. 常犯錯誤與陷阱 Common Mistakes & Traps

⚠️ 陷阱 Trap: 混淆「事實錯誤」與「價值判斷」
Confusing "Factual Errors" with "Value Judgements"

很多學生以為「錯的句子就是規範性陳述」。這是嚴重的概念錯誤！如果我說「太陽從西方升起」，這句話絕對是錯的，但它依然是實證性陳述 (Positive Statement)，因為它正是被「事實」(太陽從東方升起) 給推翻了！
Many students think "a wrong sentence is a normative statement". This is a severe conceptual error! If I say "The sun rises in the west", this statement is absolutely wrong, but it is still a Positive Statement because it is exactly refuted by the "fact" (the sun rises in the east)!

6. 學生常見疑問 Student Q&A

Q: 如果現實中，澳門的堅尼系數其實比香港還要高，那麼瑪莉的陳述還是實證性 (Positive) 嗎？
Q: If in reality, Macau's Gini coefficient is actually higher than Hong Kong's, is Mary's statement still positive?

A: 絕對是！「實證性」的定義是「可被驗證」(testable)，而不是「絕對正確」(always true)。如果澳門的數據真的比香港高，這就完美示範了瑪莉的陳述「可以被事實推翻」(can be rejected by fact)，這正是選項 C 描述的實證經濟學特徵。
A: Absolutely! The definition of "positive" is "testable", not "always true". If Macau's data is indeed higher, this perfectly demonstrates that Mary's statement "can be rejected by fact", which is exactly the characteristic of positive economics described in Option C.

Q: 要怎樣改寫彼得的句子，才能讓它變成實證性 (Positive) 陳述？
Q: How can we rewrite Peter's sentence to make it a positive statement?

A: 只要刪除主觀的「太」(too) 字，換成具體的數字即可。例如改為：「香港作為一已發展經濟體，它的堅尼系數超過了 \( 0.5 \)」。這句話就可以用數據去檢驗了。
A: Simply remove the subjective word "too" and replace it with concrete numbers. For example, change it to: "As a developed economy, Hong Kong's Gini coefficient exceeds \( 0.5 \)". This sentence can then be tested with data.

7. 考試技巧與速解 Exam Tricks & Shortcuts

一秒判定法 (The One-Second Rule):

在考場上，快速掃描句子中的形容詞和助動詞：
In the exam room, quickly scan for adjectives and auxiliary verbs in the sentence:

  • 見到 應該 (should/ought to)、公平/不公平 (fair/unfair)、好/壞 (good/bad)、太 (too) 👉 秒選 規範性 (Normative)。
    See should/ought to, fair/unfair, good/bad, too 👉 Instantly choose Normative.
  • 見到 是/不是 (is/is not)、上升/下降 (increase/decrease)、最高/最低 (highest/lowest) 👉 秒選 實證性 (Positive)。
    See is/is not, increase/decrease, highest/lowest 👉 Instantly choose Positive.

應用在此題：看到彼得說「太高」(too high)，立刻排除選項 A 和 C。看到瑪莉說「最高」(highest)，確認她是實證性，直接鎖定選項 C，完成！
Applied to this question: Seeing Peter say "too high", immediately eliminate Options A and C. Seeing Mary say "highest", confirm she is positive, directly lock in Option C. Done!
