Key Concepts (核心概念): Price-taker (受價者), Marginal Cost (邊際成本), Fixed Cost (固定成本), Profit (利潤)

1. The Question (題目)

下表顯示某受價成衣工廠的資料。該工廠的固定成本為$10。

The following table shows the data of a price-taking garment factory. The fixed cost of the factory is $10.

產出 (Output) / units 6 7 8 9 10 11
總成本 (Total cost) / $ 48 63 80 99 120 143

假設成衣的市場價格由$17上升至$19，而該工廠的固定成本則上升了$2。結果，該工廠利潤極大化的產量會 __________ ，而利潤會__________ 。

Suppose the market price of garments rises from $17 to $19, and the fixed cost of the factory rises by $2. As a result, the profit-maximising output of the factory would __________ while its profit would __________.

D. increase by 1 unit. …… increase to $82

2. Concept Review (概念重溫)

要解決這道題目，我們需要掌握以下幾個微觀經濟學的核心概念：

To solve this problem, we need to master the following core microeconomic concepts:

  • 利潤極大化條件 (Profit-Maximizing Condition): 對於受價者（完全競爭市場中的廠商），當價格 ($P$) 等於邊際成本 ($MC$) 時，利潤達到最大。
    For a price-taker (a firm in perfect competition), profit is maximized when Price ($P$) equals Marginal Cost ($MC$).
    $$P = MC$$
  • 邊際成本 (Marginal Cost, $MC$): 生產多一單位產品所增加的總成本。
    The addition to total cost from producing one more unit of output.
    $$MC = \frac{\Delta TC}{\Delta Q}$$
  • 利潤計算 (Profit Calculation): 總收入減去總成本。
    Total Revenue minus Total Cost.
    $$\pi = TR - TC = (P \times Q) - TC$$
  • 固定成本的影響 (Impact of Fixed Cost): 固定成本 ($FC$) 的改變會影響總成本 ($TC$) 和利潤 ($\pi$)，但絕不會影響邊際成本 ($MC$)。
    A change in Fixed Cost ($FC$) affects Total Cost ($TC$) and Profit ($\pi$), but it NEVER affects Marginal Cost ($MC$).

正確答案是 (Correct Option): Option C

Let's first calculate the Marginal Cost ($MC$) at each output level:

第一步：分析初始情況
初始市場價格 $P_1 = \$17$。根據 $P = MC$ 的原則，廠商會生產至 $MC = 17$ 的水平，即產量 $Q_1 = 8$。此時的利潤 $\pi_1 = TR - TC = (\$17 \times 8) - \$80 = \$136 - \$80 = \$56$。

Step 1: Analyze the initial situation
The initial market price $P_1 = \$17$. According to the $P = MC$ rule, the firm produces where $MC = 17$, which is at output $Q_1 = 8$. The initial profit $\pi_1 = TR - TC = (\$17 \times 8) - \$80 = \$136 - \$80 = \$56$.

第二步：分析變化後的情況
新市場價格 $P_2 = \$19$。固定成本上升 $\$2$ 導致每個產量下的總成本 ($TC$) 增加 $\$2$，但邊際成本 ($MC$) 維持不變。根據新的 $P = MC$，廠商會生產至 $MC = 19$ 的水平，即新產量 $Q_2 = 9$。產量由 8 上升至 9（上升1單位）。

Step 2: Analyze the new situation
The new market price $P_2 = \$19$. The fixed cost increases by $\$2$, which means the Total Cost ($TC$) at every level increases by $\$2$, but Marginal Cost ($MC$) remains unchanged. According to the new $P = MC$, the firm produces where $MC = 19$, which is at the new output $Q_2 = 9$. Output rises from 8 to 9 (increases by 1 unit).

第三步：計算新利潤
在新產量 $Q_2 = 9$ 下，原本的總成本是 $\$99$。加上新增的 $\$2$ 固定成本，新總成本 $TC' = \$99 + \$2 = \$101$。新利潤 $\pi_2 = (\$19 \times 9) - \$101 = \$171 - \$101 = \$70$。

Step 3: Calculate the new profit
At the new output $Q_2 = 9$, the original total cost was $\$99$. Adding the $\$2$ increase in fixed cost, the new total cost $TC' = \$99 + \$2 = \$101$. The new profit $\pi_2 = (\$19 \times 9) - \$101 = \$171 - \$101 = \$70$.

選項分析 (Options Breakdown):

❌ A. 維持不變 …… 上升至 $56

Incorrect. Output does not remain unchanged. It increases because the market price (and thus $MR$) increased, making it profitable to produce more. $56 was the old profit.

❌ B. 維持不變 …… 上升至 $70

Incorrect. While the profit calculation of $70 matches if we mistakenly thought output stayed at 8 (Wait, at Q=8, new TR = $19 * 8 = 152. New TC = 80 + 2 = 82. Profit = 152 - 82 = 70. Ah! This is a trap!). If the firm didn't adjust its output, profit would be $70, but a profit-maximizing firm *will* adjust output to $Q=9$ where profit is actually higher (Let's check: TR=171, TC=101, Profit=70. Wait! If output is 8, profit is $152 - 82 = 70$. If output is 9, profit is $171 - 101 = 70$. Both give $70! However, the firm expands output as long as $P \ge MC$. At $Q=9$, $P=19$ and $MC=19$. This is the profit-maximizing point. So output must increase to 9).

✅ C. 上升1單位 …… 上升至$70

Correct. The firm increases output to 9 (an increase of 1 unit). The new profit is accurately calculated as $70.

❌ D. 上升1單位 …… 上升至 $82

Incorrect. The output increase is correct, but the profit is miscalculated. $82 is the new Total Cost at $Q=8$, not the profit.

4. Problem Decomposition (題目拆解)

🔑 解題樞紐 (The Pivot Point):
這條題目的核心在於明白「固定成本 ($\Delta FC$) 的改變不會影響邊際成本 ($MC$)」。所以，當你看到固定成本上升，你只需在最後計算利潤時，把總成本加回去，完全不用重新計算 $MC$ 表格。

🔑 The Pivot Point:
The core of this question is understanding that "a change in fixed cost ($\Delta FC$) does NOT affect marginal cost ($MC$)". Therefore, when you see an increase in fixed cost, you only need to add it to the total cost at the final step of calculating profit. You don't need to recalculate the $MC$ table.

👻 擾敵資訊 (Distractors):
題目提到「工廠的固定成本為 $\$10$」。這是一個徹頭徹尾的擾敵資訊！表中的「總成本 ($TC$)」已經包含了這 $\$10$。如果你把表中的數字再加 $\$10$，你就中計了！

👻 Distractors:
The question states "The fixed cost of the factory is $\$10$". This is a complete distractor! The "Total cost ($TC$)" in the table already includes this $\$10$. If you added $\$10$ to the numbers in the table, you fell into the trap!

5. Common Mistakes & Traps (常犯錯誤與陷阱)

  • 重複計算固定成本 (Double Counting Fixed Cost): 以為表格給的是「可變成本 ($VC$)」，於是手多把 $\$10$ 加進去計算總成本。請記住，表頭寫的是「總成本 ($TC$)」。
    Thinking the table provides "Variable Cost ($VC$)" and incorrectly adding $\$10$ to find the total cost. Always read the table header carefully: it explicitly says "Total cost ($TC$)".
  • 忽略了固定成本的變化 (Ignoring the change in Fixed Cost): 在計算新利潤時，只用了舊的總成本 $\$99$，算出了 $\$171 - \$99 = \$72$ 的錯誤答案（幸好選擇題沒有這個選項）。
    Forgetting to add the $\$2$ increase in fixed cost when calculating the final profit, resulting in an incorrect profit of $\$171 - \$99 = \$72$ (luckily, not an option).

6. Student Q&A (學生常見疑問)

🤔 學生問： 為什麼我們不需要用到「固定成本為 $\$10$」這個資料？

Q: Why didn't we need to use the information "The fixed cost... is $\$10$"?

💡 導師答： 因為總成本的定義是 $TC = FC + VC$。表格提供的已經是 $TC$，這代表那 $\$10$ 已經「隱藏」在 $48, 63, 80$ 這些數字裡面了。我們只需要關心 $TC$ 的變化（即 $MC$）以及 $FC$ 後來的增幅（上升 $\$2$）。

A: Because Total Cost is defined as $TC = FC + VC$. The table already provides $TC$, which means the $\$10$ is already "baked into" the numbers $48, 63, 80$, etc. We only care about the change in $TC$ (which is $MC$) and the increase in $FC$ (the $+\$2$) later on.

🤔 學生問： 如果固定成本不是上升 $\$2$，而是大幅上升 $\$20$，產量會改變嗎？

Q: If the fixed cost increased drastically by $\$20$ instead of $\$2$, would the output change?

💡 導師答： 不會改變產量（短期內）！只要價格 $P$ 不變，$MC$ 曲線不變，利潤極大化的產量 ($P=MC$) 就不會變。但是，如果固定成本上升太多，導致出現經濟虧損，且價格低於平均可變成本 ($P < AVC$)，廠商就會選擇停業 (Shut down)。

A: The output would NOT change (in the short run)! As long as Price $P$ and the $MC$ curve remain unchanged, the profit-maximizing output ($P=MC$) stays exactly the same. However, if the fixed cost increases so much that the firm makes an economic loss AND the price falls below Average Variable Cost ($P < AVC$), the firm would choose to shut down.

7. Exam Tricks & Shortcuts (考試技巧與速解)

⏱️ 十秒殺題法 (The 10-Second Shortcut):

⏱️ The 10-Second Shortcut:

  1. 一看到 $\Delta FC$，馬上告訴自己：「$MC$ 不變！不用重計 $MC$！」
    The moment you see a change in Fixed Cost ($\Delta FC$), tell yourself: "$MC$ doesn't change! No need to recalculate $MC$!"
  2. 找新價格點：新價格是 $\$19$。查表看哪裡的相差是 $\$19$。$99 - 80 = 19$，所以新產量是 9。舊產量是 8 ($80 - 63 = 17$)。產量上升 1 單位。(馬上刪除選項 A 和 B)。
    Find the new price point: The new price is $\$19$. Look at the table for a difference of $\$19$. $99 - 80 = 19$, so new output is 9. Old output was 8 ($80 - 63 = 17$). Output increases by 1. (Instantly eliminate A and B).
  3. 計新利潤：$TR - 新TC = (\$19 \times 9) - (99 + 2) = 171 - 101 = 70$。選 C。搞定！
    Calculate new profit: $TR - \text{New } TC = (\$19 \times 9) - (99 + 2) = 171 - 101 = 70$. Choose C. Done!
