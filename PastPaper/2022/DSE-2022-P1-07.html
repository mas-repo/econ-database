下表顯示某廠商的投入、產出與成本的關係。生產所需的投入只有勞工和資本。

根據上表，我們可以得出結論：
Based on the above table, we can conclude that:

A. 該廠商享有規模經濟。
the firm enjoys economies of scale.
B. 生產的最優規模是70單位。

Concept Review (概念重溫)

1. 短期與長期 (Short Run vs. Long Run):
只要有一種投入（通常是資本 \(K\)）是固定的，該生產便處於短期。
As long as one input (usually Capital \(K\)) is fixed, the production is in the short run.

2. 平均產出 (Average Product, AP):
指平均每單位勞工所產生的產量。
Refers to the output produced per unit of labour on average.

$$AP_L = \frac{TP}{L}$$

3. 邊際產出 (Marginal Product, MP):
指增加一單位變動投入（勞工）所帶來的總產量變動。
The change in total product resulting from employing one more unit of variable input (labour).

$$MP_L = \frac{\Delta TP}{\Delta L}$$

4. 邊際回報遞減定律 (Law of Diminishing Marginal Returns, LDMR):
在短期生產中，當連續將變動投入（勞工）增加到固定投入（資本）時，邊際產出最終會下降。
In the short run, as more units of a variable input (labour) are added to a fixed input (capital), the marginal product will eventually decline.

Answer & Detailed Analysis (答案與詳細分析)

Correct Option: Option C

Option A: 錯誤 (Incorrect)
分析：規模經濟（Economies of Scale）是一個「長期」概念，前提是所有投入（包括勞工和資本）按比例增加。但在本題中，資本 \(K\) 固定在 1 單位，這屬於「短期」分析，因此無法判斷規模經濟。
Analysis: Economies of scale is a "long-run" concept where all inputs (labour and capital) increase. Since capital \(K\) is fixed at 1 unit here, this is a "short-run" analysis, making it impossible to conclude anything about economies of scale.
Option B: 錯誤 (Incorrect)
分析：「最優規模」通常指長期平均成本（LRAC）最低的產量點。本題資料不足以判斷何謂最優規模，且該表顯示的是短期數據。
Analysis: "Optimal scale" usually refers to the output level where long-run average cost (LRAC) is minimized. There is insufficient information to determine the optimal scale, and the data provided is short-run.
Option C: 正確 (Correct)
分析：我們計算每個勞工水平下的平均產出 \(AP_L\)：
- 當 \(L=1\), \(AP = 20/1 = 20\)
- 當 \(L=2\), \(AP = 50/2 = 25\)
- 當 \(L=3\), \(AP = 70/3 = 23.33\)
- 當 \(L=4\), \(AP = 80/4 = 20\)
顯然，在 \(L=2\) 時，\(AP\) 最高（25單位）。
Analysis: Let's calculate the Average Product \(AP_L\) for each level of labour:
- When \(L=1\), \(AP = 20/1 = 20\)
- When \(L=2\), \(AP = 50/2 = 25\)
- When \(L=3\), \(AP = 70/3 = 23.33\)
- When \(L=4\), \(AP = 80/4 = 20\)
Clearly, \(AP\) is highest at \(L=2\) (25 units).
Option D: 錯誤 (Incorrect)
分析：計算邊際產出 \(MP_L\)：
- 第 1 到第 2 個勞工：\(MP = 50 - 20 = 30\)
- 第 2 到第 3 個勞工：\(MP = 70 - 50 = 20\)
- 第 3 到第 4 個勞工：\(MP = 80 - 70 = 10\)
由於 \(MP\) 在增加第 3 個勞工時開始下降（由 30 跌至 20），邊際回報遞減定律適用。
Analysis: Calculate Marginal Product \(MP_L\):
- 1st to 2nd labour: \(MP = 50 - 20 = 30\)
- 2nd to 3rd labour: \(MP = 70 - 50 = 20\)
- 3rd to 4th labour: \(MP = 80 - 70 = 10\)
Since \(MP\) starts decreasing from the 3rd unit of labour onwards (from 30 to 20), the LDMR does apply.

Problem Decomposition (題目拆解)

  • Pivot Point (關鍵點): 觀察「資本 (Capital)」一欄。數字全部相同（都是 1），這立即告訴你這是短期 (Short Run) 生產問題。
  • Distractor (干擾項): 總生產成本 (Total Cost) 在這題完全是沒用的數據。它被放進去是為了誘使學生去計算平均成本或嘗試找規模經濟。
  • Approach: 看到這類表格，第一時間在旁邊加兩行：\(MP\) 和 \(AP\)。

Student Q&A (學生常見疑問)

Q: 為什麼選項 A 一定錯？如果我擴大生產規模成本真的下降了呢？

A: 在 DSE 經濟學中，概念的定義非常嚴謹。規模經濟 (Economies of Scale) 的定義是「長期平均成本隨產量增加而下降」。表格中資本固定不變，這就不是長期的情況。我們不能用短期的數據去推論長期的趨勢。即使平均成本下降，那也只是「短期平均成本 (SRAC)」的變動，不代表規模經濟。
