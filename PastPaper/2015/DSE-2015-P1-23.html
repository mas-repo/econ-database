HKDSE Economics Analysis

Topic: Economic Structure 經濟結構
Difficulty: ⭐⭐ (2/5)
Key Concepts: Employment Distribution, Absolute vs. Relative Values

1. The Question (題目)

下表顯示某經濟的就業分佈。

產業 / 年份 2010 2014
初級 0.4% 0.4%
二級 23.6% 18.5%
三級 76.0% 81.1%
整體 100% 100%
從上表我們可得出以下結論：由2010年至2014年，
A. 從事初級產業的工人數量維持不變。
B. 整體就業率是相同的。
C. 三級產業對本地生產總值的貢獻增加了。
D. 以就業而言二級產業的相對重要性減低了。
The following table shows the employment distribution in an economy.

Sector / Year 2010 2014
Primary 0.4% 0.4%
Secondary 23.6% 18.5%
Tertiary 76.0% 81.1%
Total 100% 100%
From the above table, we can conclude that from 2010 to 2014,
A. the number of workers engaged in the primary sector remained constant.
B. the overall employment rate was the same.
C. the contribution of the tertiary sector to gross domestic product (GDP) increased.
D. the relative importance of the secondary sector in terms of employment decreased.

2. Concept Review (概念重溫)

絕對數值 vs 相對數值 (Absolute vs. Relative Values)
這條題目的核心在於分辨「百分比」（相對數值）與「實際數量」（絕對數值）。表格提供的數據是就業分佈百分比，它只反映了各產業在整體就業中的相對重要性，而不能反映實際的就業人數。
Absolute vs. Relative Values
The core of this question lies in distinguishing between "percentages" (relative values) and "actual numbers" (absolute values). The table provides the percentage distribution of employment, which only reflects the relative importance of each sector, not the actual number of employed persons.
$$ \text{某產業就業人數} = \text{總就業人數} \times \text{該產業就業百分比} $$ $$ \text{Number of workers in a sector} = \text{Total employment} \times \text{Percentage share} $$
由於我們不知道「總就業人數」是否有改變，我們無法計算出實際的就業人數變化。此外，就業分佈（投入）並不等同於對 GDP 的貢獻（產出）。
Since we do not know if the "Total employment" has changed, we cannot calculate the change in the actual number of workers. Furthermore, employment distribution (input) is not equivalent to GDP contribution (output).

3. The Answer & Detailed Analysis (答案與詳細分析)

Correct Option: D

選項 A (錯誤): 表格顯示初級產業的就業百分比維持在 \(0.4\%\)。然而，如果總就業人數改變了，實際的工人數量也會隨之改變。因此，我們不能斷定工人數量維持不變。
Option A (Incorrect): The table shows the percentage share of the primary sector remained at \(0.4\%\). However, if the total employment changed, the actual number of workers would also change. Therefore, we cannot conclude the number of workers remained constant.
選項 B (錯誤): 就業率公式為 \(\frac{\text{就業人口}}{\text{勞動人口}} \times 100\%\)。表格只顯示了已就業人口內部的分佈，完全沒有提供勞動人口或失業人口的數據，因此無法得知就業率。
Option B (Incorrect): The employment rate formula is \(\frac{\text{Employed Population}}{\text{Labor Force}} \times 100\%\). The table only shows the internal distribution of the employed population and provides no data on the labor force or unemployed population.
選項 C (錯誤): 就業人數的比例增加，不代表其產值（GDP貢獻）一定增加。勞動生產力、資本投入等因素都會影響產值。
Option C (Incorrect): An increase in the proportion of employment does not necessarily mean an increase in output value (GDP contribution). Factors like labor productivity and capital input also affect output.
選項 D (正確): 二級產業的就業百分比由 \(23.6\%\) 下降至 \(18.5\%\)。「百分比」正正代表了該產業在就業方面的「相對重要性」。百分比下降，即相對重要性減低。
Option D (Correct): The employment percentage of the secondary sector dropped from \(23.6\%\) to \(18.5\%\). A "percentage" exactly represents the "relative importance" of the sector in terms of employment. A drop in percentage means a decrease in relative importance.

4. Problem Decomposition (題目拆解)

解題關鍵 (Pivot Point): 認清表格的單位是「%」。這意味著所有數據都是相對的。任何涉及「絕對數量」（如工人數目、GDP數值）的選項都可以立即被排除。
干擾資訊 (Distractors): 選項A利用了初級產業數字不變 (0.4%) 來製造陷阱，誘使學生將「百分比不變」等同於「數量不變」。
Pivot Point: Recognize that the unit in the table is "%". This means all data are relative. Any option involving "absolute quantities" (like number of workers, GDP values) can be immediately eliminated.
Distractors: Option A uses the unchanged figure for the primary sector (0.4%) as a trap, tempting students to equate "unchanged percentage" with "unchanged quantity".

5. Common Mistakes & Traps (常犯錯誤與陷阱)

  • 混淆相對與絕對數值： 這是DSE經濟科最常見的陷阱之一。緊記：百分比 \(\neq\) 實際數量。
    Confusing relative and absolute values: This is one of the most common traps in DSE Economics. Remember: Percentage \(\neq\) Actual quantity.
  • 將投入等同於產出： 投入更多勞工（就業分佈增加）不代表產出（GDP）會按比例增加。
    Equating input with output: Inputting more labor (increased employment share) does not mean output (GDP) will increase proportionally.

6. Student Q&A (學生常見疑問)

Q: 如果題目加上「假設總就業人數在2010至2014年間維持不變」，選項A會變成正確嗎？
A: 會的！如果總就業人數不變（例如維持在300萬人），那麼初級產業的工人數量就會是 \(3,000,000 \times 0.4\%\)，兩年的實際人數都會完全相同，選項A就會成立。

Q: If the question added "Assume total employment remained constant between 2010 and 2014", would Option A be correct?
A: Yes! If total employment is constant (e.g., at 3 million), the number of workers in the primary sector would be \(3,000,000 \times 0.4\%\). The actual number would be exactly the same for both years, making Option A correct.

7. Exam Tricks & Shortcuts (考試技巧與速解)

秒殺技巧 (Elimination Method):
當你看到表格只有「百分比(%)」時，立即掃描選項：
- 看到「數量」(Number) -> 刪除 (選項A)
- 看到「率」(Rate) 但缺乏分母數據 -> 刪除 (選項B)
- 看到「GDP」(產值) 但只有就業數據 -> 刪除 (選項C)
剩下的 D 就是唯一可能的答案！
Shortcut:
When you see a table with only "percentages (%)", immediately scan the options:
- See "Number" (absolute quantity) -> Eliminate (Option A)
- See "Rate" without denominator data -> Eliminate (Option B)
- See "GDP" (output) with only employment data -> Eliminate (Option C)
The remaining Option D is the only possible answer!
