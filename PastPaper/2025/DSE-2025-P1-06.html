Concept Review (概念重溫)

1. 短期 vs 長期 (Short Run vs. Long Run):

  • 短期 (Short Run): 存在固定要素 (Fixed Factors)。適用「邊際回報遞減定律」。
    Existence of Fixed Factors. "Law of Diminishing Marginal Returns" applies.
  • 長期 (Long Run): 所有要素皆為可變要素 (Variable Factors)。適用「規模經濟/不經濟」。
    All factors are Variable Factors. "Economies/Diseconomies of Scale" applies.

2. 規模不經濟 (Diseconomies of Scale):

當生產規模擴大（所有投入增加），長期平均成本 (LRAC) 上升。
When the scale of production expands (all inputs increase), Long Run Average Cost (LRAC) rises.

$$ Average\ Cost\ (AC) = \frac{Total\ Cost}{Quantity} $$

3. 規模回報遞減 (Decreasing Returns to Scale):

產出的增加比例 小於 投入的增加比例。
The percentage increase in output is less than the percentage increase in inputs.

$$ \%\Delta Output < \%\Delta Input $$

Correct Option: B

Since labour is the only input, Total Cost (TC) = Labour units \(\times\) $50.

Option B (Correct):

分析：題目指出「該廠商生產所需的投入只有勞力」。這意味著沒有固定要素，所有要素（只有勞力）都是可變的，因此這是長期 (Long Run) 生產。
當產量由 90 增至 100 時（勞力由 3 增至 4），平均成本 (AC) 由 $1.67 上升至 $2.00。長期平均成本上升定義為規模不經濟。
Analysis: The question states "The only input... is labour". This means there are no fixed factors; all factors (just labour) are variable, making this a Long Run scenario.
When output increases from 90 to 100 (labour increases from 3 to 4), Average Cost (AC) rises from $1.67 to $2.00. Rising Long Run Average Cost is defined as Diseconomies of Scale.

Option A (Incorrect):

分析：邊際回報遞減定律 (Law of Diminishing Marginal Returns) 是一個短期定律，其前提是必須有固定要素（如機器、土地）存在，導致可變要素效率下降。由於題目表明「只有勞力」是投入，即沒有固定要素，故此定律不適用。
Analysis: The Law of Diminishing Marginal Returns is a Short Run law. It requires the existence of Fixed Factors (like machines, land) which cause the variable factor's efficiency to drop. Since the question states "only input is labour", there are no fixed factors, so this law does not apply.

Option C (Incorrect):

分析：要計算虧損，我們需要比較 總收入 (TR) 和 總成本 (TC)。題目只提供了成本數據（工資），但沒有提供產品價格 (Price)，因此無法計算收入，也就無法判斷是否虧損。
Analysis: To calculate loss, we need to compare Total Revenue (TR) and Total Cost (TC). The question provides cost data (wages) but does not provide the Product Price. Without price, we cannot calculate revenue or determine if there is a loss.

Option D (Incorrect):

分析：利潤極大化條件是 \(MC = MR\)。雖然我們可以計算邊際成本 (MC)，但由於不知道價格（即不知道 MR），我們無法找出利潤極大化的產量。
Analysis: The profit-maximising condition is \(MC = MR\). Although we can calculate Marginal Cost (MC), without the price (and thus MR), we cannot identify the profit-maximising output.

Problem Decomposition (題目拆解)

1. 關鍵字眼 (The Pivot Point): "Only input... is labour"
這是解題的最關鍵線索。如果只有勞力，代表沒有固定要素 (No Fixed Factors)。
沒有固定要素 \(\rightarrow\) 長期 (Long Run)。
這直接排除了選項 A (短期概念)，並指向選項 B (長期概念)。
This is the crucial clue. If labour is the only input, there are No Fixed Factors. No Fixed Factors \(\rightarrow\) Long Run. This immediately eliminates Option A (Short Run concept) and points to Option B (Long Run concept).

2. 驗證規模回報 (Verifying Returns to Scale):
由 L=3 到 L=4：
投入增加：\((4-3)/3 = 33.3\%\)
產出增加：\((100-90)/90 = 11.1\%\)
產出增幅 < 投入增幅 \(\rightarrow\) 規模回報遞減 \(\rightarrow\) 平均成本上升 \(\rightarrow\) 規模不經濟。
Input increase: 33.3%. Output increase: 11.1%. Output growth < Input growth \(\rightarrow\) Decreasing Returns to Scale \(\rightarrow\) Rising AC \(\rightarrow\) Diseconomies of Scale.

Common Mistakes & Traps (常犯錯誤與陷阱)
  • 看見 MP 下降就選 A (Seeing MP drop and choosing A):
    這是最常見的陷阱。學生計算出 MP (20, 30, 40, 10, 5) 發現先升後跌，就直覺以為是邊際回報遞減。必須先檢查前提：是否有固定要素？ 如果沒有，就不能選 A。
    This is the biggest trap. Students calculate MP (20, 30, 40, 10, 5), see it rise then fall, and assume it's Diminishing Returns. You MUST check the condition: Are there fixed factors? If not, you cannot choose A.
