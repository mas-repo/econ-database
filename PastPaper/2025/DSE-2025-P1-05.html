📊 HKDSE Economics Topic Analysis

Topic (課題): Production Sectors & Absolute vs. Relative Values (生產階段與絕對值/相對值)

Difficulty (難度): ⭐⭐ (2/5)

Key Concepts (核心概念): Primary/Secondary/Tertiary Production (初級/次級/三級生產), Percentage Share (百分比份額), Absolute Value (絕對值)

📝 The Question (題目)

下表顯示某經濟生產總值的分佈。該經濟2024年的生產總值高於2014年。

The table below shows the distribution of total production value in an economy. The total production value in this economy was higher in 2024 than that in 2014.

2014年 (Year 2014) 2024年 (Year 2024)
初級生產 (Primary production) 15.3% 5.1%
次級生產 (Secondary production) 22.2% 22.2%
三級生產 (Tertiary production) 62.5% 72.7%
總計 (Total) 100% 100%

根據以上資料，可得出的結論是：由2014年至2024年，___________。
Based on the above information, we can conclude that from 2014 to 2024, the ____________.

  • A. 從事初級生產的工人的比例下跌了
    proportion of workers engaged in primary production decreased
  • B. 初級生產的勞工生產力下跌了
    labour productivity of primary production decreased
  • C. 次級生產的生產總值維持不變
    total production value of secondary production remained unchanged
  • D. 三級生產的生產總值必然上升了
    total production value of tertiary production must have increased

💡 Concept Review (概念重溫)

要解答這類題目，必須清楚區分相對值（百分比）和絕對值（實際金額）。題目提供的是各生產階段佔總生產值的「百分比」，而我們同時知道「總生產值」上升了。

To answer this type of question, you must clearly distinguish between relative values (percentages) and absolute values (actual amounts). The question provides the "percentage share" of each production sector, and we also know that the "total production value" has increased.

$$ \text{Sector Production Value} = \text{Total Production Value} \times \text{Percentage Share} $$ $$ \text{某部門生產值} = \text{總生產值} \times \text{所佔百分比} $$

另外，生產總值的貢獻（Value）與就業人數（Employment）或勞工生產力（Labour Productivity）是不同的概念，不能直接畫上等號。

Additionally, the contribution to total production value is a different concept from employment figures or labour productivity. They cannot be directly equated.

✅ The Answer & Detailed Analysis (答案與詳細分析)

Correct Option (正確答案): Option D

Option A Analysis (選項 A 分析): ❌

題目只提供了「生產總值」的數據，並沒有提供「就業人數」的數據。生產值佔比下降，不代表工人比例一定下降（可能是由於產品價格下跌或機器自動化等因素）。

The question only provides data on "total production value" and gives no information on "employment". A decrease in the share of production value does not necessarily mean the proportion of workers decreased (it could be due to falling product prices or automation).

Option B Analysis (選項 B 分析): ❌

勞工生產力是指平均每名工人（或每工時）的產量。由於我們缺乏實際產量（Output）和勞工投入（Labour input）的數據，因此無法判斷勞工生產力的變化。

Labour productivity refers to the output per worker (or per man-hour). Since we lack data on actual output and labour input, we cannot determine changes in labour productivity.

Option C Analysis (選項 C 分析): ❌

次級生產的百分比維持在 \(22.2\%\)。但是，由於2024年的「總生產值」高於2014年，相同的百分比乘以一個較大的總數，得出的絕對值必然是增加了，而不是維持不變。

The percentage of secondary production remained at \(22.2\%\). However, since the "total production value" in 2024 is higher than in 2014, multiplying the same percentage by a larger total sum means the absolute value must have increased, not remained unchanged.

Option D Analysis (選項 D 分析): ✅

三級生產的百分比由 \(62.5\%\) 上升至 \(72.7\%\)。同時，整體的「總生產值」也上升了。當「總數」和「所佔百分比」兩者都增加時，其絕對值（三級生產的生產總值）必然上升。

The percentage of tertiary production increased from \(62.5\%\) to \(72.7\%\). At the same time, the overall "total production value" also increased. When both the "total sum" and the "percentage share" increase, its absolute value (the total production value of tertiary production) MUST have increased.

🔍 Problem Decomposition (題目拆解)

Pivot Point (破題關鍵): 題目第一段的最後一句：「該經濟2024年的生產總值高於2014年」。這是解題的最重要線索。

Pivot Point: The last sentence of the first paragraph: "The total production value in this economy was higher in 2024 than that in 2014." This is the most crucial clue.

Distractors (干擾資訊): 表格中的「初級生產」數據。初級生產的百分比下降了 (\(15.3\% \rightarrow 5.1\%\))，但由於總數上升了，我們其實無法確定初級生產的絕對值是上升、下降還是不變（這取決於總數上升的幅度）。考評局故意用這個模稜兩可的數據來設計陷阱。

Distractors: The "Primary production" data in the table. Its percentage dropped (\(15.3\% \rightarrow 5.1\%\)), but since the total increased, we actually cannot determine whether the absolute value of primary production increased, decreased, or remained unchanged (it depends on the magnitude of the total's increase). The HKEAA deliberately uses this ambiguous data to set traps.

⚠️ Common Mistakes & Traps (常犯錯誤與陷阱)

  • 混淆百分比與絕對值 (Confusing Percentages with Absolute Values): 很多學生看到 Option C 的 \(22.2\%\) 沒有改變，就直覺認為生產值「維持不變」。忘記了基數（Base/Total）已經變大。
  • Many students see that \(22.2\%\) in Option C hasn't changed and intuitively think the production value "remained unchanged", forgetting that the base (Total) has become larger.
  • 過度推論 (Over-inference): 將「生產總值 (Production Value)」直接等同於「就業人數 (Employment)」或「實質產出 (Real Output)」。
  • Equating "Production Value" directly with "Employment" or "Real Output".

🙋‍♂️ Student Q&A (學生常見疑問)

Q: 為什麼我們不能確定初級生產的絕對值是升是跌？
Why can't we be sure if the absolute value of primary production increased or decreased?

A: 假設2014年的總生產值是 $1,000，初級生產值就是 $153 (\(1,000 \times 15.3\%\))。
如果2024年的總生產值微升至 $2,000，初級生產值就是 $102 (\(2,000 \times 5.1\%\)) ➔ 絕對值下跌。
但如果2024年的總生產值暴升至 $4,000，初級生產值就是 $204 (\(4,000 \times 5.1\%\)) ➔ 絕對值上升。
因為我們不知道總數上升了多少，所以無法確定。

Assume the total in 2014 was $1,000, primary production would be $153.
If the 2024 total slightly increased to $2,000, primary production would be $102 ➔ Absolute value decreased.
But if the 2024 total surged to $4,000, primary production would be $204 ➔ Absolute value increased.
Since we don't know the magnitude of the total's increase, it is indeterminate.

⚡ Exam Tricks & Shortcuts (考試技巧與速解)

遇到這類「總量上升 + 百分比分佈」的題目，請記住以下速解法則：

When encountering "Total Increases + Percentage Distribution" questions, remember these shortcut rules:

  • 總數上升 + 百分比上升 ➔ 絕對值必定上升 (如本題的三級生產)。
    Total Up + Percentage Up ➔ Absolute Value MUST Increase (like Tertiary here).
  • 總數上升 + 百分比不變 ➔ 絕對值必定上升 (如本題的次級生產)。
    Total Up + Percentage Unchanged ➔ Absolute Value MUST Increase (like Secondary here).
  • 總數上升 + 百分比下降 ➔ 絕對值不確定 (Unknown)。
    Total Up + Percentage Down ➔ Absolute Value is Unknown.

利用這個法則，你可以瞬間鎖定 Option D 為唯一正確答案！
