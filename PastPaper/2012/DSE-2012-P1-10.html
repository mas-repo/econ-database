概念重溫 Concept Review

勞工生產力 (Labour Productivity)： 指平均每單位勞工（通常以「工時」計算）所生產的產量。當題目同時提供「勞工人數」和「總工作時數」時，我們必須使用「總工作時數」來計算，因為這能更準確地反映實際投入的勞動量。
Labour Productivity: Refers to the output produced per unit of labour (usually measured in man-hours). When a question provides both "number of workers" and "total working hours", we must use "total working hours" as it more accurately reflects the actual labour input.
$$ \text{Labour Productivity} = \frac{\text{Total Output}}{\text{Total Working Hours}} $$
計件工資 (Piece Rate) vs 計時工資 (Time Rate)： 計件工資按產量計算薪酬，能有效鼓勵工人提高工作速度（即提高每小時產量）；計時工資按工作時間計算薪酬，工人缺乏提高生產速度的誘因。
Piece Rate vs Time Rate: Piece rate pays workers based on output, effectively incentivizing them to work faster (increasing output per hour). Time rate pays based on time spent, offering less incentive to speed up production.

答案與詳細分析 Answer & Analysis

正確答案 Correct Option: D
首先，我們需要計算兩間工廠的勞工生產力：
First, we need to calculate the labour productivity of both factories:
$$ \text{Factory A Productivity} = \frac{12000}{2000} = 6 \text{ units/hour} $$ $$ \text{Factory B Productivity} = \frac{20000}{5000} = 4 \text{ units/hour} $$
計算結果顯示，甲工廠的勞工生產力（每小時 6 單位）高於乙工廠（每小時 4 單位）。因此，我們需要找尋一個能解釋「甲工廠生產力 > 乙工廠生產力」的原因。
The calculation shows that Factory A's labour productivity (6 units/hour) is higher than Factory B's (4 units/hour). Therefore, we need to find a reason that explains why "Productivity of A > Productivity of B".
Options A & B:
如果乙工廠提供較多訓練或使用較多機器，乙工廠的生產力應該較高。這與數據（甲 > 乙）矛盾。
If Factory B provides more training or uses more machines, Factory B's productivity should be higher. This contradicts the data (A > B).
Option C:
如果甲工廠工作環境較差，甲工廠的生產力應該較低。這同樣與數據矛盾。
If Factory A's working environment is poorer, Factory A's productivity should be lower. This also contradicts the data.
Option D:
甲工廠採用計件工資，工人為了賺取更多收入會加快工作速度；乙工廠採用計時工資，工人沒有加快速度的誘因。這完美解釋了為何甲工廠的每小時產量較高。
Factory A uses piece rate, so workers speed up to earn more; Factory B uses time rate, so workers lack the incentive to speed up. This perfectly explains why Factory A has a higher output per hour.

題目拆解 Problem Decomposition

關鍵點 (Pivot Point)： 本題的關鍵在於正確計算勞工生產力。你必須識別出「聘用勞工的人數」是干擾資訊（Distractor）。如果錯誤地用產量除以人數，會得出乙工廠（2000單位/人）大於甲工廠（200單位/人）的錯誤結論。
Pivot Point: The key to this question is calculating labour productivity correctly. You must identify that "Number of workers" is a distractor. If you incorrectly divide output by the number of workers, you will reach the false conclusion that Factory B (2000 units/worker) is greater than Factory A (200 units/worker).

學生常見疑問 Student Q&A

Q: 為什麼計算生產力時不使用「聘用勞工的人數」？
Why don't we use the "number of workers" to calculate productivity?
A: 因為每個工人的工作時數可能不同。例如，甲工廠的工人可能是兼職，乙工廠的工人可能是全職。使用「總工作時數 (Man-hours)」能統一計算標準，是衡量勞動投入最準確的方法。
A: Because the working hours per worker can vary. For example, Factory A's workers might be part-time, while Factory B's are full-time. Using "Total working hours (Man-hours)" standardizes the measurement and is the most accurate way to measure labour input.

考試技巧與速解 Exam Tricks & Shortcuts

邏輯排除法 (Logical Elimination)：
在多項選擇題中，如果選項 A、B 和 C 都指向同一個結果（即提升乙工廠的生產力或降低甲工廠的生產力），那麼它們很可能都是錯的。因為如果其中一個是對的，另外兩個也可能成為合理答案。唯一指向相反結果（提升甲工廠生產力）的選項 D，必然是正確答案。這讓你在來不及計算時也能秒殺題目！
Logical Elimination:
In MCQs, if options A, B, and C all point to the same outcome (increasing B's productivity or decreasing A's), they are likely all wrong. If one were correct, the others could theoretically be correct too. Option D is the only one pointing to the opposite outcome (increasing A's productivity), making it the unique correct answer. This allows you to solve the question instantly even without calculating!
