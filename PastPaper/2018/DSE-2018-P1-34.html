HKDSE Economics Analysis

Topic: Money & Banking / Fisher Equation (貨幣與銀行 / 費雪方程式)

Difficulty: ★★★☆☆
Key Concepts: Nominal Interest Rate, Opportunity Cost

Question 題目

假設實質利率是6%，預期通脹率是3%。若實際通脹率是5%，持有貨幣的成本會是 ________。

Assume that the real interest rate is 6% and the expected inflation rate is 3%. If the actual inflation rate is 5%, the cost of holding money will be ________________.

A. 4%
B. 6%
C. 9%
D. 11%

Concept Review (概念重溫)

1. 持有貨幣的成本 (Cost of Holding Money)

持有現金（不生息資產）的機會成本，就是你放棄的利息收入。這等於市場上的名義利率 (Nominal Interest Rate)。

The opportunity cost of holding cash (non-interest-bearing asset) is the interest income foregone. This is equal to the market Nominal Interest Rate.

2. 費雪方程式 (Fisher Equation)

名義利率是在借貸合約訂立時決定的，因此它包含了實質利率和預期通脹率。

The nominal interest rate is determined when the loan contract is signed, so it incorporates the real interest rate and the expected inflation rate.

$$ R = r + \pi^e $$
  • \( R \): 名義利率 (Nominal Interest Rate)
  • \( r \): 實質利率 (Real Interest Rate)
  • \( \pi^e \): 預期通脹率 (Expected Inflation Rate)

Answer & Detailed Analysis (答案與詳細分析)

Correct Answer: Option C (9%)

Step-by-Step Solution (逐步解說)

  1. Identify the Goal: We need to find the "Cost of Holding Money", which is the Nominal Interest Rate (\(R\)).
    目標：找出「持有貨幣的成本」，即名義利率 (\(R\))。
  2. Select the Formula: Use the Fisher Equation: \( R = r + \pi^e \).
    公式：使用費雪方程式 \( R = r + \pi^e \)。
  3. Plug in Values:
    Real Interest Rate (\(r\)) = 6%
    Expected Inflation Rate (\(\pi^e\)) = 3%
    $$ R = 6\% + 3\% = 9\% $$
  4. Ignore Distractors: The Actual Inflation Rate (5%) is irrelevant for determining the nominal rate set ex-ante (beforehand).
    忽略干擾項：實際通脹率 (5%) 與事前設定的名義利率無關。
Option A (4%) - Incorrect

這可能是計算成 \( r - (\pi - \pi^e) \) 或其他隨機組合。這裡沒有經濟學根據。

This might be calculated as \( r - (\pi - \pi^e) \) or some other random combination. It has no economic basis here.

Option B (6%) - Incorrect

這只是實質利率。它忽略了補償購買力損失所需的通脹溢價。

This is just the Real Interest Rate. It ignores the inflation premium required to compensate for the loss of purchasing power.

Option C (9%) - Correct

正確計算，使用預期通脹率：\( 6\% + 3\% = 9\% \)。

Calculated correctly using Expected Inflation: \( 6\% + 3\% = 9\% \).

Option D (11%) - Common Trap!

學生常錯誤地使用實際通脹率：\( 6\% + 5\% = 11\% \)。記住，名義利率是基於預期設定的！

Students often mistakenly use the Actual Inflation Rate: \( 6\% + 5\% = 11\% \). Remember, nominal rates are set based on expectations!

Problem Decomposition (題目拆解)

The Pivot Point (關鍵點)

關鍵在於持有貨幣的成本是事前 (Ex-Ante) 決定的。銀行設定利率時無法預知未來的實際通脹，只能預測。因此，必須使用預期通脹率。

The crucial realization is that the cost of holding money is determined Ex-Ante. When banks set interest rates, they cannot know the future actual inflation; they can only guess (expect). Therefore, you MUST use Expected Inflation.

The Distractor (干擾項)

Actual Inflation Rate = 5%

這個數字的存在純粹是為了誘使你計算 \( 6 + 5 = 11 \)。它代表了這段時期之後發生的情況，這並不會改變期初設定的名義利率。

This number exists solely to trick you into calculating \( 6 + 5 = 11 \). It represents what happened after the period, which doesn't change the nominal rate set at the start.

Student Q&A (學生常見疑問)

Q: Why isn't the cost of holding money the Real Interest Rate?

問：為什麼持有貨幣的成本不是實質利率？

答：因為貨幣（現金）在通脹期間會喪失購買力。如果你持有現金，你會失去兩樣東西：

  1. 你本可以賺取的實質回報（實質利率）。
  2. 因通脹而損失的價值（預期通脹）。

將這兩者加起來就是名義利率。

A: Because money (cash) loses purchasing power during inflation. If you hold cash, you lose two things:

  1. The real return you could have earned (Real Interest Rate).
  2. The value lost due to inflation (Expected Inflation).

Adding these together gives the Nominal Interest Rate.

Q: When would we use the Actual Inflation Rate?

問：什麼時候才會用到實際通脹率？

答：我們會用實際通脹率來計算時期結束後的實現實質利率 (Realized Real Interest Rate)。

公式：\( \text{實現的 } r = \text{名義 } R - \text{實際 } \pi \)

在這條題目中，實現的 \( r = 9\% - 5\% = 4\% \)。但題目問的是持有貨幣的成本（名義利率），而不是實現的回報。

A: We use Actual Inflation to calculate the Realized Real Interest Rate after the period is over.

Formula: \( \text{Realized } r = \text{Nominal } R - \text{Actual } \pi \)

In this question, Realized \( r = 9\% - 5\% = 4\% \). But the question asked for the cost of holding money (Nominal Rate), not the realized return.
